# Quiz de Engenharia Social

Um jogo educativo interativo sobre segurança digital e engenharia social, desenvolvido para conscientizar usuários sobre golpes e técnicas de manipulação online.

## Sobre o Projeto

Este quiz apresenta 5 perguntas fundamentais sobre engenharia social, ajudando usuários leigos a entenderem melhor os riscos de segurança digital e como se proteger contra golpes online.

### Temas Abordados

- Conceitos básicos de engenharia social
- Identificação de alvos e técnicas de manipulação
- Golpes de phishing
- Boas práticas de segurança
- O fator humano na segurança digital

## Tecnologias Utilizadas

- **HTML5** - Estrutura do jogo
- **CSS3** - Estilização e animações
- **JavaScript** - Lógica do quiz e interatividade

## Características

- Interface moderna com tema escuro (preto e verde neon)
- Sistema de pontuação com feedback imediato
- Animações suaves e responsivas
- Barra de progresso visual
- Design responsivo para desktop e mobile
- Efeitos visuais de destaque nas respostas corretas/incorretas

## Como Executar

### Opção 1: Abrir diretamente no navegador

1. Faça o download dos arquivos do projeto
2. Extraia os arquivos em uma pasta
3. Abra o arquivo `index.html` diretamente no seu navegador

### Opção 2: Usando Live Server no VS Code (Recomendado)

1. Abra o Visual Studio Code
2. Instale a extensão "Live Server"
   - Vá em Extensões (Ctrl+Shift+X)
   - Busque por "Live Server"
   - Clique em Install
3. Abra a pasta do projeto no VS Code
4. Clique com o botão direito no arquivo `index.html`
5. Selecione "Open with Live Server"

## Estrutura de Arquivos

```
├── index.html      # Estrutura HTML do jogo
├── styles.css      # Estilos e animações
```

## Sistema de Pontuação

- **5 acertos**: "Excelente! Você é um expert em segurança!"
- **3-4 acertos**: "Muito bem! Você está no caminho certo."
- **1-2 acertos**: "Você precisa estudar mais sobre segurança digital."
- **0 acertos**: "Atenção! É importante aprender sobre engenharia social."

## Respostas Corretas

1. **O que é engenharia social?**
   - Resposta: A) Um tipo de golpe que usa a manipulação das pessoas

2. **Qual é o principal alvo da engenharia social?**
   - Resposta: B) As pessoas e seu comportamento

3. **Um golpe de phishing normalmente tenta fazer você:**
   - Resposta: B) Clicar em links falsos ou revelar informações pessoais

4. **Qual atitude ajuda a evitar golpes de engenharia social?**
   - Resposta: C) Desconfiar de mensagens estranhas e verificar a fonte

5. **Por que o fator humano é considerado o elo mais fraco da segurança?**
   - Resposta: A) Porque as pessoas podem ser enganadas e manipuladas

## Customização

Você pode facilmente customizar o quiz editando:

- **Perguntas**: Modifique o array `questions` no arquivo `script.js`
- **Cores**: Ajuste as variáveis CSS no arquivo `styles.css`
- **Mensagens**: Altere os textos de feedback na função `showResults()` em `script.js`
